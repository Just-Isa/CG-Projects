Single letters represent the buttons on a keyboard

Changing colours:
        Background      :   S, R, G, B
        Object          :   SHIFT + S, SHIFT + R, SHIFT + G, SHIFT + B,  

Moving Object           :   Hold rightclick, and move the mouse accordingly

Scaling object          :   - Hold the mouse wheel, and drag to the right to scale up, and to the left to scale down
                            - Scroll up / down with the mouse wheel

Rotating object         :   - Hold leftclick, and move the mouse accordingly

Switch Perspective: 
        Orthogonal      :   O
        Perspective     :   P

Shadows                 :   - Press j to activate/deactivate
